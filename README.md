## Fabric SPA - Test Task

This Laravel 8/Vue 3 SPA using the REST API

## Features

-   PHP 8
-   Laravel 8
-   Vue 3 + Vuex
-   SPA starts
-   Bootstrap 5
-   Webpack

## Installation

-   `git clone`
-   Edit `.env` and set your database connection details and **your APP_URL**
-   `php artisan key:generate`
-   `npm install`
-   `composer install`
-   `npm run dev`

## Usage

#### Development

npm run watch
